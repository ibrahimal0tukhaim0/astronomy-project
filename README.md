# Islamic Astronomy Simulation

An interactive 3D solar system simulation that combines scientific accuracy with Islamic astronomical heritage.

## Features
- **Solar System Simulation**: Accurate 3D representation of planets and orbits.
- **Islamic Integration**: Quranic verses, Tafsir, and historical context for celestial bodies.
- **Bilingual**: Full support for English and Arabic (RTL).
- **Stunning Visuals**: High-quality materials, glows, and orbital trails.

## Recent Updates (Version 1.2)
- **Refined Orion's Belt**: Split the belt into three distinct stars (Alnitak, Alnilam, Mintaka) with accurate positioning and spacing.
- **Enhanced Visuals**: Improved star visibility and alignment.
- **Localization**: Added deep historical and religious context for Orion's stars.
